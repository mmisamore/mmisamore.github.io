Found [this post](http://christopherpoole.github.io/using-mathjax-on-github-pages)
which help me to set up mathjax support that was compatible with github. In
summary, I switched to the kramdown parser in Jekyll, dropped the Mathjax CDN
code into the head tag of my default.html, and rewrote 
[Fun with Finite Strings](/2011/02/20/Fun-with-finite-strings) to use real
LaTeX between \\\\( and )\\\\ tags. I'm really enjoying these markdown-style
languages.
